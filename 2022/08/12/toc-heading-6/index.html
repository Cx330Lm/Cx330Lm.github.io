0x001 PHP 表单

1. HTML 表单

<!DOCTYPE html>
<html>
<body>
    <form action="index.php" method="post">
        账号：<input name="username" type="username"></br>
        密码：<input name="password" type="password"></br>
        <input name="提交" type="submit">
</body>
</html>

当用户填写此表单并点击提交按钮后，表单数据会发送到名为 “index.php” 的 PHP 文件供处理。表单数据是通过 HTTP POST 方法发送的。

GET 方式和 POST方式的区别：

  • $_GET 是通过 URL 参数传递到当前脚本的变量数组。
  • $_POST 是通过 HTTP POST 传递到当前脚本的变量数组。

获取提交的内容信息：

<html>
<body>
    账号：<?php echo $_POST["username"];?>
    密码：<?php echo $_POST["password"];?>
</body>
</html>

GET 方式的表单实现方式和 POST 差不多，只需把 POST 替换为 GET 方式就可以

2. PHP 表单验证

先写一个简单的 PHP 表单

<!DOCTYPE html>
<html>
    <meta charset="UTF-8">
    <title>PHP 表单</title>
<body>
    <form method="POST" action="<?php echo htmlspecialchars($_SERVER['PHP_SELF']);?>">
        姓名：<input type="name" name="name"></br></br>

        邮箱：<input type="email" name="email"></br></br>

        网址：<input type="text" name="url"></br></br>

        性别：<input type="radio" name="gender" value="female">男
             <input type="radio" name="gender" value="male">女</br></br>

        评论：<textarea name="comment" rows="5" cols="40"></textarea></br></br>

        提交：<input type="submit" name="提交">
    </form>
</body>
</html>

$_SERVER["PHP_SELF"] 是一种超全局变量，它返回当前执行脚本的文件名。

因此，$_SERVER["PHP_SELF"] 将表单数据发送到页面本身，而不是跳转到另一张页面。这样，用户就能够在表单页面获得错误提示信息。

htmlspecialchars() 函数把特殊字符转换为 HTML 实体。这意味着 < 和 > 之类的 HTML 字符会被替换为 &lt; 和 &gt; 。这样可防止攻击者通过在表单中注入 HTML 或 JavaScript 代码（跨站点脚本攻击）对代码进行利用。

<form method="POST" action="<?php echo $_SERVER["PHP_SELF"];?>">

假设我们的 index.php 中有如下表单，当输入正常网址 http://www.xxx.com/index.php 时，上面的代码就会转换为如下：

<form method="POST" action="index.php">

但是当恶意用户输入这样的 http://www.xxx.com/index.php/"><script>alert('XSS')</script> 网址时

上面的代码就会变成

<form method="POST" action="index.php/"><script>alert('XSS')</script>

此时页面就会出现一个 XSS 弹窗，从而获取用户的登录信息。

怎样避免这种情况的发生呢？使用 htmlspecialchars() 函数可以将特殊字符进行转义，从而使代码无法正常被执行。此时上面的代码就会变成如下：

<form method="post" action="index.php/&quot;&gt;&lt;script&gt;alert('XSS')&lt;/script&gt;">

接下来写一个完整的实例：

<?php
// 自定义白能量并设置为空值
$name = $email = $gender = $comment = $url ="";
if($_SERVER['REQUEST_METHOD'] == "POST"){
    $name = test_input($_POST["name"]); 
    $email = test_input($_POST["email"]); 
    $url = test_input($_POST['url']);
    $gender = test_input($_POST['gender']);
    $comment = test_input($_POST['comment']);
}

function test_input($data){
    $data = trim($data); // 首尾去空
    $data = stripcslashes($data); // 删除由 addcslashes() 函数添加的反斜杠。
    $data = htmlspecialchars($data); // 对特殊字符进行转义
}
?>

4. PHP 验证URL/email

PHP 验证名字

$name = test_input($_POST["name"]);
if (!preg_match("/^[a-zA-Z ]*$/",$name)) {
  $nameErr = "只允许字母和空格！"; 
}

注释：preg_match() 函数检索字符串的模式，如果模式存在则返回 true，否则返回 false。

PHP 验证 email

$email = test_input($_POST["email"]);
if (!preg_match("/([\w\-]+\@[\w\-]+\.[\w\-]+)/",$email)) {
  $emailErr = "无效的 email 格式！"; 
}

PHP 验证 URL

$website = test_input($_POST["website"]);
if (!preg_match("/\b(?:(?:https?|ftp):\/\/|www\.)[-a-z0-9+&@#\/%?=~_|!:,.;]*[-a-z0-9+&@#\/%
=~_|]/i",$website)) {
  $websiteErr = "无效的 URL"; 
}

现在写一个完整的验证 名字、email 和 URL 的代码

<?php
// 定义变量并设置为空值
$nameErr = $emailErr = $genderErr = $urlErr = "";
$name = $email = $gender = $comment = $url = "";

if ($_SERVER["REQUEST_METHOD"] == "POST") {
  if (empty($_POST["name"])) {
    $nameErr = "名字不能为空";
  } else {
    $name = test_input($_POST["name"]);
    // 检查名字是否包含字母和空格
    if (!preg_match("/^[a-zA-Z ]*$/",$name)) {
      $nameErr = "只允许字母和空格"; 
    }
  }

  if (empty($_POST["email"])) {
    $emailErr = "邮箱不能为空";
  } else {
    $email = test_input($_POST["email"]);
    // 检查电邮地址语法是否有效
    if (!preg_match("/([\w\-]+\@[\w\-]+\.[\w\-]+)/",$email)) {
      $emailErr = "邮箱格式无效"; 
    }
  }

  if (empty($_POST["url"])) {
    $url = "";
  } else {
    $url = test_input($_POST["url"]);
    // 检查 URL 地址语言是否有效（此正则表达式同样允许 URL 中的下划线）
    if (!preg_match("/\b(?:(?:https?|ftp):\/\/|www\.)[-a-z0-9+&@#\/%?=~_|!:,.;]*[-a-z0-9+&@#\/%
    =~_|]/i",$url)) {
      $urlErr = "无效的网址"; 
    }
  }

  if (empty($_POST["comment"])) {
    $comment = "";
  } else {
    $comment = test_input($_POST["comment"]);
  }

  if (empty($_POST["gender"])) {
    $genderErr = "性别不能为空";
  } else {
    $gender = test_input($_POST["gender"]);
  }
}
?>

0x002 PHP 文件处理

PHP readfile() 函数

readfile() 函数读取文件，并把它写入输出缓冲。

<?php
    echo readfile("file.txt");
?>

1. PHP 文件打开/读取/关闭

PHP 打开文件

打开文件的函数是 fopen()，fopen() 的第一个参数包含被打开的文件名，第二个参数规定打开文件的模式。

<?php
$myfile = fopen("file.txt", "r") or die("Unable to open file!");
echo fread($myfile,filesize("file.txt"));
fclose($myfile);
?>

fread() 以及 fclose() 函数。

模式 描述
r 打开文件为只读。文件指针在文件的开头开始。
w 打开文件为只写。删除文件的内容或创建一个新的文件，如果它不存在。文件指针在文件的开头开始。
a 打开文件为只写。文件中的现有数据会被保留。文件指针在文件结尾开始。创建新的文件，如果文件不存在。
x 创建新文件为只写。返回 FALSE 和错误，如果文件已存在。
r+ 打开文件为读/写、文件指针在文件开头开始。
w+ 打开文件为读/写。删除文件内容或创建新文件，如果它不存在。文件指针在文件开头开始。
a+ 打开文件为读/写。文件中已有的数据会被保留。文件指针在文件结尾开始。创建新文件，如果它不存在。
x+ 创建新文件为读/写。返回 FALSE 和错误，如果文件已存在。

PHP 读取文件

fread() 函数读取打开的文件。

fread() 的第一个参数包含待读取文件的文件名，第二个参数规定待读取的最大字节数。

fread($myfile,filesize("file.txt"));

PHP 关闭文件

fclose() 函数用于关闭打开的文件。

注释：用完文件后把它们全部关闭是一个良好的编程习惯。您并不想打开的文件占用您的服务器资源。

fclose() 需要待关闭文件的名称（或者存有文件名的变量）：

<?php
$myfile = fopen("file.txt", "r");
// 需要执行的代码
fclose($myfile);
?>

PHP 读取单行文件

<?php
$myfile = fopen("file.txt", "r") or die("Unable to open file!");
echo fgets($myfile);
fclose($myfile);
?>

注释：调用 fgets() 函数之后，文件指针会移动到下一行。

PHP 检查

feof() 函数检查是否已到达 "end-of-file" (EOF)。

feof() 对于遍历未知长度的数据很有用。

<?php
$myfile = fopen("file.txt", "r") or die("Unable to open file!");
// 输出单行直到 end-of-file
while(!feof($myfile)) {
  echo fgets($myfile) . "<br>";
}
fclose($myfile);
?>

PHP 读取单字符

fgetc() 函数用于从文件中读取单个字符。

<?php
$myfile = fopen("file.txt", "r") or die("Unable to open file!");
// 输出单字符直到 end-of-file
while(!feof($myfile)) {
  echo fgetc($myfile);
}
fclose($myfile);
?>

2. PHP 创建/写入文件

PHP 创建文件

fopen() 打开并不存在的文件，此函数会创建文件，假定文件被打开为写入 w 或增加 a。

$myfile = fopen("file.txt", "w")

如果运行这段代码时发生错误，请检查是否有向硬盘写入信息的 PHP 文件访问权限。

PHP 写入文件

fwrite() 的第一个参数包含要写入的文件的文件名，第二个参数是被写的字符串。

<?php
$myfile = fopen("file.txt", "w") or die("Unable to open file!");
$txt = "Hello\n";
fwrite($myfile, $txt);
$txt = "World\n";
fwrite($myfile, $txt);
fclose($myfile);
?>

PHP 覆盖

<?php
$myfile = fopen("file.txt", "w") or die("Unable to open file!");
$txt = "hello world\n";
fwrite($myfile, $txt);
$txt = "Hello World\n";
fwrite($myfile, $txt);
fclose($myfile);
?>

3. PHP 文件上传

先写一个文件上传表单

<!DOCTYPE html>
<html>
    <meta charset="UTF-8">
    <title>PHP 文件上传表单</title>
<body>
    <form action="upload.php" method="post" enctype="multipart/form-data">
        <label for="file">文件名称:</label>
        <input type="file" name="file" id="file" /> <br />
        <input type="submit" name="submit" value="提交" />
    </form>
</body>
</html>

创建上传脚本

<?php
if ($_FILES["file"]["error"] > 0)
  {
  echo "Error: " . $_FILES["file"]["error"] . "<br />";
  }
else
  {
  echo "Upload: " . $_FILES["file"]["name"] . "<br />";
  echo "Type: " . $_FILES["file"]["type"] . "<br />";
  echo "Size: " . ($_FILES["file"]["size"] / 1024) . " Kb<br />";
  echo "Stored in: " . $_FILES["file"]["tmp_name"];
  }
?>

参数解释：

  • $_FILES[“file”][“name”] - 被上传文件的名称
  • $_FILES[“file”][“type”] - 被上传文件的类型
  • $_FILES[“file”][“size”] - 被上传文件的大小，以字节计
  • $_FILES[“file”][“tmp_name”] - 存储在服务器的文件的临时副本的名称
  • $_FILES[“file”][“error”] - 由文件上传导致的错误代码

上传限制(对文件格式进行限制，只允许上传 .gif 和 .jpg 格式的文件)

<?php

if ((($_FILES["file"]["type"] == "image/gif")
|| ($_FILES["file"]["type"] == "image/jpeg")
|| ($_FILES["file"]["type"] == "image/pjpeg"))
&& ($_FILES["file"]["size"] < 20000))
  {
  if ($_FILES["file"]["error"] > 0)
    {
    echo "Error: " . $_FILES["file"]["error"] . "<br />";
    }
  else
    {
    echo "Upload: " . $_FILES["file"]["name"] . "<br />";
    echo "Type: " . $_FILES["file"]["type"] . "<br />";
    echo "Size: " . ($_FILES["file"]["size"] / 1024) . " Kb<br />";
    echo "Stored in: " . $_FILES["file"]["tmp_name"];
    }
  }
else
  {
  echo "Invalid file";
  }

?>

注释：对于 IE，识别 jpg 文件的类型必须是 pjpeg，对于 FireFox，必须是 jpeg。

保存被上传的文件

<?php
if ((($_FILES["file"]["type"] == "image/gif")
|| ($_FILES["file"]["type"] == "image/jpeg")
|| ($_FILES["file"]["type"] == "image/pjpeg"))
&& ($_FILES["file"]["size"] < 20000))
  {
  if ($_FILES["file"]["error"] > 0)
    {
    echo "Return Code: " . $_FILES["file"]["error"] . "<br />";
    }
  else
    {

    echo "Upload: " . $_FILES["file"]["name"] . "<br />";
    echo "Type: " . $_FILES["file"]["type"] . "<br />";
    echo "Size: " . ($_FILES["file"]["size"] / 1024) . " Kb<br />";
    echo "Temp file: " . $_FILES["file"]["tmp_name"] . "<br />";

    if (file_exists("upload/" . $_FILES["file"]["name"]))
      {
      echo $_FILES["file"]["name"] . " already exists. ";
      }
    else
      {
      move_uploaded_file($_FILES["file"]["tmp_name"],
      "upload/" . $_FILES["file"]["name"]);
      echo "Stored in: " . "upload/" . $_FILES["file"]["name"];
      }
    }
  }
else
  {
  echo "Invalid file";
  }
?>

这里脚本会检测是否已存在此文件，如果不存在，则把文件拷贝到指定的 upload 文件夹下。

0x003 PHP cookies 和 sessions

1. cookies

cookie 常用于识别用户。cookie 是服务器留在用户计算机中的小文件。每当相同的计算机通过浏览器请求页面时，它同时会发送 cookie。

创建 cookie

setcookie() 函数用于设置 cookie。

注释：setcookie() 函数必须位于 标签之前。

语法：

setcookie(name, value, expire, path, domain);

<?php 
setcookie("user", "pikachu", time()+3600);
?>

<html>
<body>

</body>
</html>

注释：在发送 cookie 时，cookie 的值会自动进行 URL 编码，在取回时进行自动解码（为防止 URL 编码，请使用 setrawcookie() 取而代之）。

获取 cookie 值

PHP 的 $_COOKIE 变量用于获取 cookie 的值。

<?php
// 输出对应用户的 cookie 值
echo $_COOKIE["user"];

// 查看所有 cookie 值
print_r($_COOKIE);
?>

使用 isset() 函数来确认是否已设置了 cookie：

<html>
<body>

<?php
if (isset($_COOKIE["user"]))
  echo "Welcome " . $_COOKIE["user"] . "!<br />";
else
  echo "Welcome guest!<br />";
?>

</body>
</html>

删除 cookie

删除 cookie 时，使过期日期变更为过去的时间点。

<?php 
// 将日期设置为一小时前
setcookie("user", "", time()-3600);
?>

浏览器不支持 cookie 的解决办法

如果应用程序涉及不支持 cookie 的浏览器，就不得不采取其他方法在应用程序中从一张页面向另一张页面传递信息。一种方式是从表单传递数据。

<html>
<body>
    <form action="index.php" method="post">
        姓名: <input type="text" name="name" />
        年龄: <input type="text" name="age" />
        <input type="submit" />
    </form>
</body>
</html>

获取 index.php 中的值：

<html>
<body>
    姓名： <?php echo $_POST["name"]; ?>.<br />
    年龄： <?php echo $_POST["age"]; ?> 
</body>
</html>

2. sessions

Session 的工作机制是：为每个访问者创建一个唯一的 id (UID)，并基于这个 UID 来存储变量。UID 存储在 cookie 中，亦或通过 URL 进行传导。

开启 session

注释：session_start() 函数必须位于 标签之前：

<?php session_start(); ?>

<html>
<body>

</body>
</html>

上面的代码会向服务器注册用户的会话，以便您可以开始保存用户信息，同时会为用户会话分配一个 UID。

存储 Session 变量

<?php
session_start();
// 存储会话数据
$_SESSION['views']=1;
?>

<html>
<body>

<?php
// 检索会话数据
echo "Pageviews=". $_SESSION['views'];
?>

</body>
</html>

创建一个简单的 page-view 计数器。isset() 函数检测是否已设置 “views” 变量。如果已设置 “views” 变量，我们累加计数器。如果 “views” 不存在，则我们创建 “views” 变量，并把它设置为 1：

<?php
session_start();

if(isset($_SESSION['views']))
  $_SESSION['views']=$_SESSION['views']+1;

else
  $_SESSION['views']=1;
echo "Views=". $_SESSION['views'];
?>

结束 Session

如果希望删除某些 session 数据，可以使用 unset() 或 session_destroy() 函数。

unset() 函数用于释放指定的 session 变量：

<?php
unset($_SESSION['views']);
?>

也可以通过 session_destroy() 函数彻底终结 session：

<?php
session_destroy();
?>

注释：session_destroy() 将重置 session，您将失去所有已存储的 session 数据。
